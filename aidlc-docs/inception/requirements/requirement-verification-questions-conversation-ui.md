## Q1: シナリオパネルのトグルボタン

現在の要件では、ゴールパネル（🎯）とペルソナパネル（👤）にはトグルボタンが定義されていますが、シナリオパネルには個別のトグルボタンが定義されていません。

シナリオパネルの表示/非表示はどのように制御しますか？

- A) シナリオパネルは常に表示（トグル不要）
- B) ペルソナパネルと連動（👤ボタンでペルソナ+シナリオを同時トグル）
- C) シナリオパネル専用のトグルボタンをヘッダーに追加する
- D) 右側パネル全体（ゴール+シナリオ+ペルソナ）を一括トグルするボタンを用意する
